# Marginale Belastingdruk

Interactive marginal tax rate calculator for the Netherlands (2026).

**[Live demo](https://geerto97.github.io/marginale-belastingdruk/)**

## Features

- Marginal and effective tax rate visualization with interactive chart
- Monthly-to-yearly salary converter with vakantiegeld (8% holiday allowance) toggle
- Breakdown of tax components: income tax brackets, algemene heffingskorting, arbeidskorting
- Collapsible methodology section explaining all calculations
- NL/EN language toggle
- Dark mode
- Fully responsive

## Tech stack

React 19, Vite, Tailwind CSS v4, Recharts

## Getting started

```bash
npm install
npm run dev
```
